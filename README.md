# 🎓 Diktat-App für die Schule

Kostenlose Web-App für Schul-Diktate mit Text-to-Speech.

**Live:** https://mycollecto.github.io/diktat-app/

## Features
- 🔊 Text-to-Speech – Browser liest den Text vor
- 🌍 Mehrsprachig – Deutsch, Englisch, Französisch, Spanisch, Italienisch
- ⏱️ Einstellbare Lesegeschwindigkeit
- 🔤 Zeichensetzung wird angesagt
- 👁️ Text ausblenden während des Diktats
- 📱 PWA – auf Homescreen installierbar
- 💾 Kein Login, keine Daten gespeichert

## Einsatz
Ideal für:
- Schüler die zuhause Diktate üben
- Lehrer die Diktate vorbereiten
- Sprachlernende aller Altersstufen

## Setup
GitHub Pages: Settings → Pages → Branch: main

*Keine Installation nötig – läuft direkt im Browser*
