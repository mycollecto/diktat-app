# 🎙️ Diktat-App

**Automatisches Diktat für die Schule – direkt im Browser, installierbar auf iPhone & Android.**

## Was die App kann

- 📋 **Text einfügen** – beliebigen Diktat-Text eintippen oder einfügen
- 📷 **Foto aufnehmen** – Diktat abfotografieren, Text wird automatisch erkannt
- 🎙️ **Vorlesen** – jeden Satz bis zu 4× wiederholen, Satzzeichen werden mitdiktiert
- ⚙️ **Einstellbar** – Tempo, Wiederholungen und Schreibpause frei wählbar
- 📵 **Offline-fähig** – funktioniert nach dem ersten Laden auch ohne Internet

## Foto-Texterkennung

Die Erkennung läuft komplett auf dem Gerät via **Tesseract.js** – kostenlos, kein Account, keine Daten werden hochgeladen.

## App installieren

**iPhone / iPad:** Link in Safari öffnen → Teilen (□↑) → „Zum Home-Bildschirm"

**Android:** Link in Chrome öffnen → Menü ⋮ → „App installieren"

## Technologie

Progressive Web App (PWA) – kein Framework, kein Build-Tool, kein App Store.
Text-zu-Sprache via Web Speech API. Texterkennung via Tesseract.js.
